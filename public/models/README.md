# Fitting Room avatar

`mannequin.glb` is the neutral body used in the 3D Fitting Room.

## Provenance & license

It is the **MakeHuman base mesh** (the "body" material group only — all
helper/clothing/joint geometry stripped), re-centred and scaled to ~1.7 m,
exported to glTF.

The MakeHuman **application** is AGPL, but its **base mesh and target assets
are released under CC0** by the MakeHuman project — precisely so that meshes
produced from it are unencumbered. Only that CC0 mesh data is included here;
none of the MakeHuman/`makehuman-js` application code ships with the app.

- MakeHuman: https://www.makehumancommunity.org/
- Asset license: CC0 1.0 (public domain)

Regenerated from the `makehuman-data` package's base model; the extraction
script is not committed (it is a one-off build step, not part of the app).
